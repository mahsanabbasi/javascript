History Of JavaScript

JavaScript was created by Brendan Eich in 1995, during his time at Netscape. Originally developed in just ten days, JavaScript was initially called "Mocha" and then renamed "LiveScript" before finally being branded as "JavaScript" to capitalize on the growing popularity of Java. Although JavaScript and Java share similarities in name and some syntax, they are very different languages. In 1997, JavaScript was standardized by the European Computer Manufacturers Association (ECMA) as ECMAScript, ensuring that it followed a set of guidelines that browsers could implement consistently. Over the years, ECMAScript has undergone numerous revisions, adding new features and improvements that make JavaScript more powerful and user-friendly than ever before.
